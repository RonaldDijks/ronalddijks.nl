# ronalddijks.nl

A shot at a online portfolio / blogging website. I want to write about things that I encounter in programming and music, so here is my little slice of internet where this all can live.

## Overview

- `pages/blog/*` Blog posts that make use of the MDX format.
- `pages/*` All other singleton pages.

## Running localy

```bash
$ git clone https://github.com/RonaldDijks/ronalddijks.nl
$ cd ronalddijks.nl
$ yarn
$ yarn dev
```

## Technologies used

- [Next.JS](https://www.nextjs.org)
- [Vercel](https://www.vercel.com)
- [MDX](https://github.com/mdx-js/mdx)
- [Chakra UI](https://chakra-ui.com/)
